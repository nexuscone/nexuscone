# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is published at that URL and applies in all project spaces, including the GitHub repository, issue tracker, discussions, and any public channels run on behalf of the project.

## Reporting

To report a concern, please contact the maintainer at **`osi@aperintel.com`** with the subject line `[Nexuscone Conduct]`. Reports are kept confidential. You can expect an acknowledgement within 72 hours and a determination on next steps within seven days.

If your concern is about the maintainer, please raise it via the GitHub project page's report mechanism instead.

## Scope

This applies to anyone participating in the project, including maintainers, contributors, sponsors, and visitors interacting with project resources.
